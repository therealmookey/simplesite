# SimpleSite - Website Builder

Maak eenvoudig je eigen website zonder technische kennis!

## ✨ Features

- 🎨 **Drag & Drop** blokken toevoegen
- ✏️ **Bewerk** content direct op de pagina
- 🖼️ **Upload** je eigen afbeeldingen
- 🔗 **Voeg knoppen** met links toe
- 📦 **Download** als complete website (ZIP)
- 💳 **Betaalsimulatie** (geen echte betaling)
