# SimpleSite - Website Builder

Maak eenvoudig je eigen website met een stappenplan en lagen-systeem!

## ✨ Features

- 📋 **Stappenplan** - Doorloop 4 simpele stappen
- 🎨 **Lagen-systeem** - Bouw je pagina's laag voor laag (zoals Photoshop)
- 📄 **Meerdere pagina's** - Kies welke pagina's je wilt
- ✏️ **Bewerk content** - Klik en typ direct op de pagina
- 🖼️ **Upload afbeeldingen** - Eenvoudig afbeeldingen toevoegen
- 📦 **Download als ZIP** - Complete website bestanden
- 💳 **Betaalsimulatie** - Demo betaling
